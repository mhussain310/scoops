# Getting Started

1. Clone this repository in your local system. 
2. Head over to [https://newsapi.org/](https://newsapi.org/) and get your own API Key. 
3. Open the cloned repository in VS Code or another code editor of your choice.  
4. Open a terminal and run `npm install` to install the dependencies.  
5. Create a `.env` file in the root directory.  
6. Inside the `.env` file, type the following: `REACT_APP_API_KEY=INSERT_YOUR_API_KEY` (do not include the square brackets).  
7. Save the changes and then, in your terminal, type `npm start` to view the application.
